The content of the directories of this project:

- input: contains the used dataset before cleaning
- output: contains the cleaned dataset obtained after EDA notebook and used in the rest of the notebooks
- notebook: contains the jupyter notebooks developed in the project
- notebook_html: the same previous notebooks but in html format
